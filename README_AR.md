# الباحث الاستشعاري — نسخة GitHub Pages

هذه النسخة جاهزة للرفع على GitHub Pages.

## الملفات التي ترفعها إلى المستودع
- index.html
- manifest.webmanifest
- service-worker.js

## الخطوات
1. أنشئ مستودعًا جديدًا في GitHub.
2. ارفع الملفات الثلاثة إلى جذر المستودع.
3. من Settings > Pages فعّل GitHub Pages.
4. اختر النشر من الفرع الرئيسي Main ومن المجلد Root.
5. انتظر حتى يظهر لك رابط GitHub Pages.

## التجربة على الآيفون
1. افتح الرابط من Safari على iPhone.
2. اضغط مشاركة.
3. اختر Add to Home Screen.
4. افتح التطبيق من الشاشة الرئيسية.

## ملاحظات
- التطبيق يحفظ البيانات محليًا داخل المتصفح على نفس الجهاز.
- إذا فتحت نفس الرابط على جهاز آخر فلن تظهر نفس البيانات السابقة.
- إذا أردت لاحقًا أيقونة حقيقية للتطبيق وSplash screen أفضل، يمكن إضافتها.
